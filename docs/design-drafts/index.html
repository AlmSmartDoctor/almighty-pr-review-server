주력 후보 2안 — Split vs Drill

같은 데이터(멀티벤더 PR findings 트리아지)를 두 가지 내비게이션 철학으로 그린 v2 초안입니다. 둘 다 라이트 테마 · 상단 레포 탭(전체+4레포) · 오버뷰(상태·리뷰필요성 배지·severity 요약) · 디테일(리뷰 트레이스 타임라인 + findings 트리아지 + 구조화 코멘트 프리뷰)을 완전 오프라인으로 담았습니다. 차이는 디테일에 어떻게 도달하느냐 하나입니다.

선택 포인트: 목록과 상세를 동시에 보며 빠르게 오가려면 Split, 한 PR에 몰입해 넓은 화면으로 트리아지하려면 Drill. 상세 비교는 COMPARISON.md 참고.
▍ v2 주력 후보
주력light2-pane
Variant 1 · Master–Detail

Split

한 화면 2-pane. 왼쪽 PR 목록에서 고르면 오른쪽 디테일이 즉시 갱신 — 컨텍스트를 잃지 않고 여러 PR을 훑는다.

  • 좌 마스터: PR별 상태·리뷰필요성·severity 요약 목록 + 검색/필터
  • 우 디테일: 리뷰 트레이스 타임라인 · findings 트리아지 · 코멘트 프리뷰
  • 선택해도 화면 전환 없음 — 목록 위치 유지
  • 승인/기각 토글 시 진행 카운터·프리뷰 실시간 반영
내비
마스터–디테일 (전환 없음)
밀도
High — 목록+상세 동시 조망
강점
다수 PR 빠른 비교·연속 트리아지
Split 열기 ./variant-split.html
주력lightfull-screen
Variant 2 · Drill-Down

Drill

오버뷰 ↔ 디테일 전체화면 이동. 상황판에서 PR을 누르면 넓은 상세로 들어가고, esc/뒤로로 복귀 — 한 번에 하나에 몰입.

  • 오버뷰: KPI 상황판 + PR 카드(상태·리뷰필요성·severity·합의)
  • 디테일: 전체화면 리뷰 트레이스 · findings 트리아지 · 프리뷰
  • 카드 클릭으로 진입, 뒤로가기(오버뷰 버튼/esc)로 복귀
  • 디테일에서 승인/기각 → 오버뷰 배지에 반영
내비
전체화면 드릴다운 + 뒤로가기
밀도
Medium — 상세는 넓게, 몰입
강점
복잡한 PR 정밀 트리아지·낮은 산만도
Drill 열기 ./variant-drill.html
